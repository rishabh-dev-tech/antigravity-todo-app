## What I Built

I built a professional Todo Task Manager web application where users can add tasks, mark them as completed using a checkbox, and delete tasks when they are finished. The application has a clean UI with a gradient background and modern task cards.

---

## Why I Built It

The goal of this project was to demonstrate frontend development skills using HTML, CSS, and JavaScript while creating a simple and effective task management interface.

---

## Features

- Add tasks
- Add tasks by pressing **Enter**
- Mark tasks as completed with a **checkbox**
- Completed tasks become transparent with **line-through**
- Delete tasks
- Scrollable task list
- Clean and modern UI
